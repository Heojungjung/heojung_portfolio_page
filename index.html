프로젝트 소개

프론트엔드 국비지원 교육 과정 과제로 '정부 공공기관 웹사이트 리뉴얼' '혁신24' 사이트 리디자인 작업을 했습니다.

작업 유형 : 개인 작업 작업 기간 : 24.08.07 ~ 24.10.24
작업 환경 : HTML5 / CSS3 / JavaScript / Swiper.js 라이브러리 사용

1

프로젝트 목표

혁신 24 기존 디자인(24.10 기준)
'혁신 24'를 리뉴얼 대상으로 선정한 이유는 기존 사이트의 디자인이 상당히 노후화 되었다 생각하여
'이 페이지를 작업하면 Before / After 비교가 확실하겠다!' 란 생각이 들었기 때문입니다.

작업에 있어 주요 목표로 잡은 점은 이렇습니다.

1. 타 정부기관 사이트들을 벤치마킹하여 디자인의 통일성을 갖추기.

2. 기존 사이트의 지나치게 많은 텍스트와 요소 덜어내기.

3. 무의미하게 세분화된 페이지 및 불필요한 페이지들 하나의 페이지로 통합하기.

KRDS 사이트

그 과정에서 적극 참고한 사이트가 바로
'KRDS' 라는 사이트입니다.

'KRDS'는 정부사이트로, 저와 생각이 비슷한 분이 정부쪽에 있었는지 '각 공공사이트 디자인의 통일'을 목적으로 UI/UX 가이드라인을 배포하는 사이트입니다.
저는 해당 사이트를 적극 참조하여 컴포넌트를 제작하였습니다.

'서울 어린이 대공원'을 리뉴얼 대상으로 선정한 이유는 이전 작업 사이트인 정부 사이트와 다른 디자인도 가능하다는 것을 보여주고 싶었던 점이 큽니다.
개인 작업인 혁신24 리뉴얼이 정적이면서 심플한 디자인이었다면 이번 작업물은 동적이며 아기자기한 디자인을 추구했습니다!

작업에 있어 주요 목표로 잡은 점은 이렇습니다.

1. '어린이' '대공원'에 초첨을 맞춰 동적이고 아기자기한 디자인을 갖추기.

2. 날씨 API / 지도 API를 적절하게 활용하여 정보 제공하기.

3. 외부사이트로의 이동을 최소화하며 하나의 페이지에서 예매 기능 구현하기.

경주월드 사이트

이번 작업에 있어선 테마파크 사이트들을 적극 참고하였고 그 중 메인 레퍼런스로 삼은 사이트는
'경주월드' 사이트입니다.

'경주월드' 사이트는 저희 사이트가 필요로 하는 기능들을 공통되게 갖고 있었으며 디자인 또한 매우 잘 되어 있어 많은 영감을 받았습니다.
